Αποτελει ακαδημαική εργασια
Οδηγίες για πλήρη λειτουργία της εφαρμογής
Back end file
1.	Στο αρχείο server.js να οριστεί η κατάλληλη πόρτα
2.	Στο αρχείο .env να οριστούν τ στοιχεία σύνδεσης με της βάσης για το php admin (localhost)
Front end file
1.	Στο αρχείο App.js να τοποθετηθεί η τοπική ip του υπολογιστή και η πόρτα
2.	Src/api/client.js θα τοποθετηθεί η ip του υπολογιστή
3.	Src/screens/CategoryShowScreen.js να τοποθετηθεί η τοπική ip του υπολογιστή και η πόρτα
4.	Src/screens/ShowDetailsScreen.js να τοποθετηθεί η τοπική ip του υπολογιστή και η πόρτα
5.	Src/screens/ShowsFromCarousel.js να τοποθετηθεί η τοπική ip του υπολογιστή και η πόρτα


Να δημιουργήσετε λογαριασμό στο php admin και να δώσετε όλα τα δικαιώματα για να τα εκχωρήσετε στο αρχείο .env
